# Ayre e-shop

Primera version de tienda online estatica para Ayre.

## Abrir el sitio

Abri `index.html` en el navegador.

## Incluye

- Portada visual con identidad inicial de Ayre.
- Catalogo con busqueda y filtro por categoria.
- Carrito lateral con cantidades, total y persistencia local.
- Diseno responsive para desktop y mobile.
